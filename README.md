# valentinios

## Run locally

1. Start server:

```bash
node server.js
```

2. Open site:

- Main page: `http://localhost:3000/`
- Admin panel: `http://localhost:3000/new`

## What `/new` shows

- visitor IP
- country / city / region (best effort by IP)
- device type
- browser and OS
- visited page path
- referrer
- screen and viewport size
- language and timezone
